# 💰 Finance Vision

Blog acadêmico de educação financeira desenvolvido com React + Vite para a atividade AOP2.

O projeto foi criado com foco em:
- estrutura semântica HTML;
- responsividade;
- componentização;
- navegação SPA;
- design moderno;
- boas práticas de frontend.

---

# 📚 Sobre o Projeto

O **Finance Vision** é um blog de educação financeira que apresenta conteúdos sobre:
- investimentos;
- controle financeiro;
- planejamento financeiro;
- educação financeira;
- organização de gastos.

O sistema possui uma interface moderna, organizada e responsiva, simulando um blog profissional de finanças.

---

# 🎯 Objetivo Acadêmico

Este projeto foi desenvolvido para atender os requisitos da atividade **AOP2**, contendo:

✅ Estrutura HTML semântica  
✅ Navegação entre páginas  
✅ Home com destaques  
✅ Página de contatos  
✅ Responsividade  
✅ Organização visual e proporcionalidade  
✅ Utilização de React + Vite

# 🖥️ Páginas do Projeto

O blog possui 5 páginas principais:

| Página | Descrição |
|---|---|
| Home | Página inicial com resumo e destaques |

# 🚀 Como Executar o Projeto

1. Clone o repositório:
```bash
git clone https://github.com/lucas-cristian/finance-vision.git
```

2. Acesse a pasta do projeto:
```bash
cd finance-vision
```

3. Instale as dependências:
```bash
npm install
```

4. Execute o projeto:
```bash
npm run dev
```

5. Acesse no navegador:
```
http://localhost:5173
```
